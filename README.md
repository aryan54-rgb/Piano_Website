# The Royal Court Piano 🎹

A beautifully styled, interactive medieval-themed piano web application with authentic organ sounds, keyboard support, and recording capabilities.

## Features

✨ **Interactive Piano Keyboard**
- 2 octaves of playable keys (visual display)
- Keyboard shortcuts for convenient playing
- Mouse click and touch screen support
- Visual feedback with active key highlighting

🎵 **Authentic Audio**
- Pipe organ character via additive synthesis
- Multiple harmonics for warm, medieval tone
- Built-in hall reverb and echo effects
- Smooth attack, decay, and release envelope

🎙️ **Recording & Playback**
- Record your compositions in real-time
- Play back recorded melodies
- Track recording status with visual indicator

🌙 **Ambient Soundscape**
- Medieval hall drone with low frequencies
- Filtered ambient noise for immersion
- Toggleable ambience with visual indicator

🕯️ **Medieval Aesthetics**
- Flickering candle animations
- Period-accurate design with gold and parchment colors
- Responsive layout for desktop and mobile
- Decorative elements and typography

⌨️ **Keyboard Bindings**
- **Lower Octave:** A S D F G H J (natural notes) + W E T Y U (sharps)
- **Upper Octave:** K L ; ' (natural notes) + O P (sharps)
- Octave adjustment with UI buttons

🎚️ **Controls**
- Octave up/down adjustment (1-7)
- Volume slider
- Record/Play functionality
- Ambience toggle

## How to Play

1. **Via Keyboard:** Press the mapped keys (A, S, D, F, etc.) to play individual notes
2. **Via Mouse/Touch:** Click or tap the piano keys directly on screen
3. **Adjust Octave:** Use the arrows to shift between octaves
4. **Volume Control:** Use the slider to adjust master volume

## Recording

1. Click the **Record** button to start capturing your performance
2. Play your melody using the keyboard or mouse
3. Click **Stop** to finish recording
4. Click **Play** to replay your composition

## Ambience

Toggle the **Ambience** button to add a medieval hall atmosphere with low drone frequencies and filtered noise.

## Technical Details

- **Audio Synthesis:** Web Audio API with oscillators and gain nodes
- **Envelope:** ADSR-style attack, decay, sustain, and release
- **Reverb:** Simple delay feedback for hall effect
- **Visualizer:** Dynamic frequency visualization bars
- **Styling:** CSS Grid, Flexbox, gradients, and animations
- **Fonts:** Cinzel (serif), IM Fell English (italic serif)

## Browser Compatibility

Works in all modern browsers with Web Audio API support:
- Chrome/Edge
- Firefox
- Safari
- Opera
